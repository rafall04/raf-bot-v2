# RAF Bot V2 - Product Overview

## What It Is
WhatsApp-based customer management system for an ISP (Internet Service Provider) business. Combines a WhatsApp bot with a web admin panel for managing subscribers, billing, support tickets, and network monitoring.

## Core Features
- **Customer Management**: PPPoE user provisioning, subscription packages, billing cycles
- **WhatsApp Bot**: Automated customer support, payment notifications, service requests
- **Billing System**: Invoice generation (PDF), payment tracking, auto-isolation for unpaid accounts
- **Support Tickets**: Customer issue reporting with technician workflow
- **Network Monitoring**: MikroTik router integration, GenieACS TR-069 device management
- **Agent System**: Reseller/agent management with voucher sales and balance tracking

## Business Context
- Indonesian ISP business (RAF NET)
- Timezone: Asia/Jakarta
- Currency: Indonesian Rupiah (IDR)
- Language: Indonesian (Bahasa Indonesia) for user-facing content

## Key Integrations
- WhatsApp via Baileys library (multi-device)
- MikroTik RouterOS API
- GenieACS (TR-069 ACS)
- iPaymu payment gateway
